# OpenPI 入门

OpenPI 是面向机器人学习的开源项目，可用于研究视觉、语言与动作模型（VLA）。

## 学习路线

1. 准备 Python、CUDA 与机器人数据集环境。
2. 理解观测、语言指令与动作序列的数据结构。
3. 运行预训练模型推理。
4. 使用自有数据进行微调与评估。
